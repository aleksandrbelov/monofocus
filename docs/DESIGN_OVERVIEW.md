# DESIGN_OVERVIEW

**Concept:** Ultra-minimal monochrome timer. Big digits, one primary action.

**User Flow:**
- Launch → Set preset or custom time → Start
- Optional: Tap Run next to DND/Grayscale shortcuts
- At end: gentle haptic + local notification tone

**Accessibility:**
- Dynamic Type and monospaced digits
- High contrast by default
- VoiceOver labels on controls
