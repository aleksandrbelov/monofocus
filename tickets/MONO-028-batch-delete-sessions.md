# MONO-028: Batch Delete Sessions

**Type:** Feature  
**Priority:** 🟢 Medium  
**Status:** Backlog  
**Estimate:** 1 hour  
**Sprint:** 4 (User Experience Enhancements)

---

## 📝 Description

Allow users to select multiple sessions in edit mode and delete them in one action — useful for clearing test sessions or removing an entire day's history.

---

## 🎯 Acceptance Criteria

- [ ] "Edit" button in History view top-right activates multi-select mode
- [ ] Selected rows show a checkmark; tapping toggles selection
- [ ] "Select All" button appears in edit mode
- [ ] "Delete N Sessions" toolbar button is enabled only when ≥ 1 row is selected
- [ ] Confirmation alert before deleting: "Delete 3 sessions? This cannot be undone."
- [ ] After delete, chart and stats update immediately
- [ ] Accessible: edit mode announcement via VoiceOver
